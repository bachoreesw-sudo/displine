1️⃣ ลงทะเบียน / เข้าสู่ระบบ
• กด ลงทะเบียน → กรอกข้อมูลให้ครบ
• รหัสผ่านอย่างน้อย 6 ตัว
• กด 👁 ดู/ซ่อนรหัสผ่าน
2️⃣ วางแผน
• เลือกระยะเวลา → ใส่เป้าหมาย
• กด ➕ เพิ่มกิจกรรม → ใส่ชื่อ เวลา เป้าหมาย นาที
• เปิด 🔁 ซ้ำทุกวัน → กด 💾 บันทึก
3️⃣ เริ่มภารกิจ
① กดเริ่มต้นเลย
② นับถอยหลัง 60 วิ
③ จับเวลาตามแผน | ⚡ Pop-up ตรวจ 3 ครั้ง
④ กิจกรรมเสร็จ → ⏭ ถัดไป
⑤ ✅ เสร็จภารกิจทั้งหมด
4️⃣ แจ้งเตือนก่อนเวลา 2 นาที
• ระบบจะแจ้งเตือนอัตโนมัติก่อนถึงเวลาแผน 2 นาที
• ตัวอย่าง: แผน 19:00 น. → แจ้งเตือน 18:58 น.
• ✅ ต้องอนุญาต Notification ในเบราว์เซอร์
• หน้าหลักจะแสดง badge 🔴 ถ้ายังไม่ได้อนุญาต
• กด ⚙ ตั้งค่า → บันทึก Discord ด้วย
5️⃣ Discord Webhook ⚠️
① Discord → ห้อง → ⚙ ตั้งค่า
② Integrations → Webhooks → New
③ Copy URL → วางในหน้าตั้งค่า
