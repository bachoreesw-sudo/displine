✦ دَلِيلُ الاسْتِخْدَام ✦
1️⃣ การสมัครและเข้าสู่ระบบ
• กด ลงทะเบียน → กรอกชื่อผู้ใช้ รหัสผ่าน ชื่อ นามสกุล ชั้น และอนาคตที่ฝันไว้
• รหัสผ่านต้องมีอย่างน้อย 6 ตัวอักษร
• กด 👁 เพื่อดูหรือซ่อนรหัสผ่านได้
• ข้อมูลถูกบันทึกลงในอุปกรณ์ของคุณและ Firebase (ถ้าตั้งค่าแล้ว)
2️⃣ การวางแผน (📋 Plan)
• เลือกระยะเวลา: 1 สัปดาห์ / 1 เดือน / 1 ปี
• ใส่เป้าหมายรวมของแผน เช่น "สอบ ONET ให้ได้ 80+"
• กด ➕ เพิ่มกิจกรรม → ใส่ชื่อ เวลาเริ่ม-สิ้นสุด เป้าหมาย จำนวนนาที
• เปิดสวิตช์ 🔁 ซ้ำทุกวัน เพื่อใช้แผนเดิมทุกวัน
• กด 💾 บันทึกแผน เพื่อยืนยัน
3️⃣ เริ่มภารกิจ (🚀 Start)
ขั้นตอน:
① กด เริ่มต้นเลย → ดูแผนวันนี้ แก้ไขได้ก่อนเริ่ม
② หน้า นับถอยหลัง 60 วิ → อ่านอายัตกุรอาน เตรียมใจ หรือกดข้ามได้
③ หน้า จับเวลา → นับถอยหลังตามแผน
   ⚡ จะมี Pop-up ถาม 3 ครั้ง ว่ายังทำอยู่ไหม (สุ่มเวลา) ต้องกดยืนยันภายใน 30 วิ
④ เมื่อเหลือ 1 นาที → อัปโหลดสรุป (URL รูป หรือข้อความ) ไม่บังคับ
⑤ กด ✅ เสร็จภารกิจ! → บันทึก + ส่งแจ้งเตือน Discord
4️⃣ ดูประวัติ (📅 History)
• ปฏิทินแสดงวันที่ทำสำเร็จด้วย ● สีเขียว
• กดวันใดก็ได้เพื่อดูสรุปรายละเอียด
• วันที่ทำสำเร็จจะมีพื้นหลังสีเขียวอ่อน
5️⃣ ตั้งค่า Icon และข้อความ (🎨)
เลือก Icon: ต้นไม้ / บ้าน / เมือง / รถ
 → Icon จะเจริญตามความคืบหน้า วงแหวนทองรอบ Icon = % ที่ทำสำเร็จ

สไตล์ข้อความหน้าจอ:
🤲 ให้กำลังใจ — ข้อความอ่อนโยน บวก
📖 แบบอิสลาม — อ้างอิงอัลกุรอาน/ฮาดิษ
😤 ด่า — กระตุ้นแบบตรงๆ
🔥 ด่าเจ็บแสบ — สำหรับคนที่ต้องการแรงกระตุ้นสูง ⚠️

สไตล์แจ้งเตือน Discord — ข้อความที่ส่งไป Discord เมื่อไม่ทำภารกิจ
6️⃣ ตั้งค่า Discord Webhook ⚠️ สำคัญมาก
Discord Webhook จำเป็นต้องใส่ เพื่อรับการแจ้งเตือนทุกวัน

วิธีสร้าง Webhook URL:
① เปิด Discord → เข้าห้องที่ต้องการรับแจ้งเตือน
② กด ตั้งค่าห้อง (⚙) → เมนู Integrations
③ กด Webhooks → New Webhook
④ ตั้งชื่อ (เช่น "แบยี") → กด Copy Webhook URL
⑤ วาง URL ที่ได้ในหน้า ⚙ ตั้งค่า → ช่อง Discord Webhook
⑥ กด 💾 บันทึก และ 🧪 ทดสอบ ว่าได้รับข้อความหรือไม่

⚠️ ถ้าไม่ใส่ Discord Webhook การแจ้งเตือนจะไม่ทำงาน
7️⃣ Firebase — บันทึกข้อมูลออนไลน์
ข้อมูลทั้งหมด (ประวัติ, แผน, บัญชี) จะถูกบันทึกใน Firebase ของแอดมิน
ไม่ต้องตั้งค่าเอง ข้อมูลจะซิงค์ข้ามอุปกรณ์ได้อัตโนมัติ

ถ้าต้องการ Firebase ส่วนตัว:
ไปที่ ⚙ ตั้งค่า → 🔥 Firebase Configuration → ใส่ค่าจาก Firebase Console ของคุณ
แล้วกด 🚀 เชื่อมต่อ Firebase
8️⃣ ความหมายของ Icon
🌳 ต้นไม้ — ทำสำเร็จ = เขียวสด, ล้มเหลว = เหี่ยวแห้ง 🍂
🏠 บ้าน — ทำสำเร็จ = บ้านสวยงาม, ล้มเหลว = บ้านพัง 💥
🏙 เมือง — ทำสำเร็จ = เมืองเจริญ, ล้มเหลว = เมืองถูกถล่ม 🔥
🚗 รถ — ทำสำเร็จ = รถใหม่วิ่งดี, ล้มเหลว = รถพัง 💢

วงแหวนทอง รอบ Icon = % ความคืบหน้าของแผนทั้งหมด
🔥 Streak = จำนวนวันที่ทำต่อเนื่อง
